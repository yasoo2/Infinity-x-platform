## 1. Product Overview
بطاقة تسجيل دخول بسيطة تربط بين صفحة الهبوط ولوحة التحكم. تحتوي على حقول البريد الإلكتروني وكلمة المرور مع زر عين لإظهار/إخفاء كلمة المرور، وخيار تذكرني، مع تصميم جميل ومختلف.

المنتج يهدف إلى توفير تجربة تسجيل دخول سلسة للمستخدمين مع الحفاظ على الأمان وسهولة الاستخدام.

## 2. Core Features

### 2.1 User Roles
لا يوجد تمييز بين الأدوار في هذه المرحلة - جميع المستخدمين لديهم نفس صلاحيات تسجيل الدخول إلى لوحة التحكم.

### 2.2 Feature Module
نظام تسجيل الدخول يتكون من المكونات التالية:
1. **بطاقة تسجيل الدخول**: نموذج إدخال البريد الإلكتروني وكلمة المرور، زر عين لإظهار/إخفاء كلمة المرور، خيار تذكرني.
2. **نظام المصادقة**: إرسال طلب POST إلى /api/v1/auth/login مع إمكانية الرجوع إلى GET عند الحاجة.
3. **إدارة التوكن**: تخزين التوكن محلياً في المتصفح لحفظ حالة تسجيل الدخول.
4. **إعادة التوجيه**: تحويل المستخدم إلى صفحة /dashboard بعد نجاح تسجيل الدخول.

## 3. Core Process
يتدفق المستخدم من صفحة الهبوط إلى لوحة التحكم من خلال عملية تسجيل دخول بسيطة:

1. المستخدم ينقر على زر تسجيل الدخول في صفحة الهبوط
2. تظهر بطاقة تسجيل الدخول كطبقة فوقية أو في صفحة منفصلة
3. يدخل المستخدم بريده الإلكتروني وكلمة المرور
4. يمكنه استخدام زر العين لرؤية/إخفاء كلمة المرور
5. يختار خيار "تذكرني" إذا رغب في حفظ بياناته
6. ينقر على زر تسجيل الدخول
7. يتم إرسال الطلب إلى الخادم
8. عند النجاح، يتم تخزين التوكن وتوجيه المستخدم إلى لوحة التحكم

```mermaid

    A[صفحة الهبوط] --> B[بطاقة تسجيل الدخول]
    B --> C{إدخال البيانات}
    C --> D[زر تسجيل الدخول]
    D --> E[POST /api/v1/auth/login]
    E -->|نجاح| F[تخزين التوكن]
    E -->|فشل| G[عرض خطأ]
    F --> H[/dashboard]
    G --> C
```

## 4. User Interface Design

### 4.1 Design Style
- الألوان الأساسية: أزرق داكن (#1a365d) للخلفية، أبيض للبطاقة، أزرق فاتح (#3182ce) للأزرار
- أسلوب الأزرار: زوايا مستديرة مع تأثيرات ظل خفيفة
- الخطوط: خط نظيف حديث مثل Inter أو Tajawal، حجم 16px للنصوص الرئيسية
- أسلوب التنسيق: بطاقة مركزية مع ظلال ناعمة وتأثيرات انتقال سلسة
- الرموز: استخدام أيقونة عين بسيطة من مكتبة React Icons

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| بطاقة تسجيل الدخول | الحاوية الرئيسية | بطاقة بيضاء مستديرة الزوايا، عرض 400px، ظل خفيف، خلفية متدرجة |
| بطاقة تسجيل الدخول | حقل البريد الإلكتروني | حقل إدخال طويل، حدود رمادية فاتحة، تركيز أزرق |
| بطاقة تسجيل الدخول | حقل كلمة المرور | حقل إدخال مع زر عين في الجهة اليمنى، أيقونة متغيرة |
| بطاقة تسجيل الدخول | خيار تذكرني | مربع اختيار صغير مع نص جانبي |
| بطاقة تسجيل الدخول | زر التسجيل | زر كبير ملون، تأثير hover، عرض كامل |
| بطاقة تسجيل الدخول | رسائل الخطأ | نص أحمر صغير أسفل الحقول المرتبطة |

### 4.3 Responsiveness
تصميم يبدأ من سطح المكتب مع تكيف تلقائي للأجهزة المحمولة. عرض البطاقة ثابت عند 400px على الشاشات الكبيرة ويصبح مرنًا (90%) على الشاشات الصغيرة.

### 4.4 التفاعل والحركات
- تأثير انتقال ناعم عند ظهور البطاقة (fade in + scale)
- تأثير اهتزاز بسيط عند ظهور رسائل الخطأ
- تغيير لون حدود الحقول عند التركيز
- تأثير نبضي خفيف عند النقر على زر التسجيل
