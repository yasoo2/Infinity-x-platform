## 1. نظرة عامة على المنتج
نظام تسجيل دخول كامل يتيح للمستخدمين الوصول إلى لوحة التحكم من صفحة الهبوط. يتميز بنموذج بسيط للبريد الإلكتروني وكلمة المرور مع مصادقة JWT وإعادة التوجيه التلقائي إلى لوحة التحكم.

النظام مصمم لتوفير تجربة دخول سلسة مع حماية آمنة للتوكن وإمكانية تغيير عنوان API حسب الحاجة.

## 2. الميزات الأساسية

### 2.1 أدوار المستخدمين
| الدور | طريقة التسجيل | الصلاحيات الأساسية |
|------|----------------|------------------|
| مستخدم عادي | التسجيل بالبريد الإلكتروني | تصفح لوحة التحكم الأساسية |
| مستخدم مميز | ترقية عبر لوحة التحكم | وصول كامل لجميع الميزات |

### 2.2 وحدات الميزات
المتطلبات الأساسية للنظام تتكون من الصفحات التالية:
1. **صفحة الهبوط**: قسم البطل، نموذج تسجيل الدخول، روابط التنقل.
2. **صفحة لوحة التحكم**: واجهة المستخدم الرئيسية بعد تسجيل الدخول، معلومات الحساب.
3. **صفحة جو**: الصفحة المخصصة للمستخدم جو بعد تسجيل الدخول.

### 2.3 تفاصيل الصفحات
| اسم الصفحة | اسم الوحدة | وصف الميزة |
|-----------|-------------|-------------|
| صفحة الهبوط | قسم البطل | عرض رسالة ترحيبية مع شعار النظام |
| صفحة الهبوط | نموذج تسجيل الدخول | حقول إدخال البريد الإلكتروني وكلمة المرور مع زر تسجيل الدخول |
| صفحة الهبوط | روابط التنقل | روابط للصفحات المهمة مثل التسجيل واستعادة كلمة المرور |
| لوحة التحكم | معلومات الحساب | عرض اسم المستخدم وبريده الإلكتروني وتاريخ آخر تسجيل دخول |
| لوحة التحكم | قائمة التنقل | شريط جانبي أو أعلى يحتوي على روابط الأقسام المختلفة |
| صفحة جو | الواجهة المخصصة | محتوى مخصص للمستخدم جو مع بياناته الشخصية |

## 3. العملية الأساسية
يتم تسجيل الدخول من خلال إدخال البريد الإلكتروني وكلمة المرور في صفحة الهبوط. بعد التحقق من صحة البيانات، يتم إنشاء توكن JWT وتخزينه في المتصفح، ثم يتم إعادة التوجيه تلقائياً إلى صفحة /dashboard/joe.

```mermaid
graph TD
  A[صفحة الهبوط] --> B{نموذج تسجيل الدخول}
  B --> C[التحقق من البيانات]
  C --> D[إنشاء توكن JWT]
  D --> E[حفظ التوكن]
  E --> F[/dashboard/joe]
  F --> G[صفحة جو]
```

## 4. تصميم واجهة المستخدم

### 4.1 نمط التصميم
- الألوان الأساسية: أزرق (#2563EB) وأبيض (#FFFFFF)
- الألوان الثانوية: رمادي فاتح (#F3F4F6) ورمادي غامق (#6B7280)
- نمط الأزرار: مستديرة الحواف مع ظل خفيف
- الخط: Inter أو Tajawal بحجم 16px للنص الأساسي
- نمط التخطيط: تخطيط عمودي مركزي مع بطاقات منفصلة
- الأيقونات: أيقونات خطية بسيطة من Heroicons

### 4.2 نظرة عامة على تصميم الصفحات
| اسم الصفحة | اسم الوحدة | عناصر واجهة المستخدم |
|-----------|-------------|-------------|
| صفحة الهبوط | قسم البطل | خلفية متدرجة من الأزرق إلى الأبيض، شعار مركزي، عنوان كبير بحجم 48px |
| صفحة الهبوط | نموذج تسجيل الدخول | بطاقة بيضاء بعرض 400px، حقول إدخال بارتفاع 48px، زر أزرق كبير |
| لوحة التحكم | معلومات الحساب | بطاقة مع معلومات المستخدم، صورة افتراضية دائرية، إحصائيات سريعة |
| صفحة جو | الواجهة المخصصة | تخطيط شبكي مع أقسام مختلفة، ألوان مخصصة حسب تفضيلات المستخدم |

### 4.3 التجاوب
المنتج مصمم بشكل أساسي للكمبيوتر المكتبي مع قابلية التكيف مع الأجهزة المحمولة. يتم تحسين التفاعل للمس مع أحجام أزرار مناسبة (48px كحد أدنى) وتباعد كافي بين العناصر.
