1. تيرمنال 1:
   cd FutureSystems_full_project/backend
   node server.mjs

   بعدها جرّب بالرَوْزِر:
   http://localhost:4000/system/health

2. تيرمنال 2:
   cd FutureSystems_full_project/worker
   BACKEND_URL="http://localhost:4000" node brain.mjs

   لازم تشوف بالكونسول كل ~15 ثانية "pulse -> OK"
   يعني جو (العامل) عم يرسل نبض للبكند.
