# سجل قدرات نظام JOEngine

هذا المستند يوثق القدرات الأساسية والمتقدمة لنظام JOEngine. هذا ليس مجرد خادم ويب، بل هو منصة تطوير ذكاء اصطناعي متكاملة وخلاقة.

---

## 1. الخالق: محرك الذكاء الاصطناعي المتقدم (`ai-engine.service.mjs`)

هذا هو المحرك الإبداعي للنظام، المسؤول عن توليد وتحسين الكود باستخدام Google Gemini 1.5 Pro. إنه "الفنان" الذي يبني الحلول.

### القدرات الرئيسية:

*   **توليد مواقع ويب كاملة:**
    *   القدرة على إنشاء مواقع ويب احترافية ومتكاملة من وصف نصي باللغة العربية.
    *   **يشمل:** تصميم مستجيب، SEO، أداء عالي، وإمكانية الوصول.
    *   يدعم `Tailwind CSS` وأيقونات حديثة.

*   **تحسين الكود مع حماية ذكية:**
    *   يستخدم "عقدًا صارمًا" مع الذكاء الاصطناعي لضمان عدم حذف أو تخريب الكود الأصلي.
    *   **آلية حماية (`Smart Merge`):** في حال فشل الـ AI، يتم دمج الكود الجديد بذكاء بدلاً من استبدال الكود الأصلي.

*   **تحويل اللغات البرمجية:**
    *   القدرة على "ترجمة" الكود من لغة إلى أخرى (e.g., Python to JS).

*   **تحليل الكود المدعوم بالذكاء الاصطناعي:**
    *   يستطيع تقديم تحليل شامل للكود (نقاط قوة، ضعف، اقتراحات) باستخدام الذكاء الاصطناعي.

---

## 2. العقل: أدوات الأكواد الذكية (`code.tool.mjs`)

هذا هو المحرك التحليلي للنظام، المسؤول عن فحص وتدقيق الكود. إنه "الناقد" الذي يضمن الجودة.

### القدرات الرئيسية:

*   **محرك تحليل ثابت متقدم (SAST):**
    *   **كشف الثغرات الأمنية:** SQL Injection, XSS, Hardcoded Secrets, و Command Injection.
    *   **تقييم جودة الكود:** حساب درجة جودة وتقدير (A+, B, etc.).
    *   **قياس التعقيد:** حساب التعقيد الدوري ومستوى التداخل.

*   **نظام تحسين الكود (Optimizer):**
    *   **تقليل الحجم (Minification):** لزيادة سرعة التحميل.
    *   **تحسين الأداء:** تحسينات محددة مثل تحسين الحلقات.

*   **مولّد الأكواد (Code Generator):**
    *   القدرة على كتابة مقتطفات كود بناءً على مواصفات (e.g., API Clients, React Components).

---

## 3. اليد العاملة: مدير الأدوات المتقدمة (`AdvancedToolsManager.mjs`)

هذا هو المحرك التنفيذي الذي يوفر بيئة معزولة (Sandbox) لتنفيذ المهام العملية.

### القدرات الرئيسية:

*   **تنفيذ كود ديناميكي:**
    *   تنفيذ أكواد Python و Node.js بشكل آمن ومعزول.

*   **معالجة وتحليل البيانات:**
    *   تحليل CSV و JSON، وإجراء تحليلات إحصائية.

*   **معالجة الوسائط المتعددة:**
    *   إنشاء رسوم بيانية وتعديل الصور.

*   **إدارة الملفات:**
    *   التحويل بين الصيغ، والضغط، وفك الضغط.
